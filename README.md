# Dummy Crawl Board

This folder is a static website for scraping practice.

## What is included

- 50 board items total
- 3 paginated pages (`index.html`, `page2.html`, `page3.html`)
- Simple static HTML and CSS only (crawler friendly)

## GitHub Pages quick deploy

1. Create a repository named `<your-username>.github.io`.
2. Upload all files from this folder to the repository root.
3. Commit and push.
4. Visit `https://<your-username>.github.io/`.
